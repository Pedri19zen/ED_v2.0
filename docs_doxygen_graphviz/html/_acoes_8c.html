◆ AbrirNovaCaixa()

int AbrirNovaCaixa ( Supermercado * S)

Requisito 5: abre uma caixa.

Requisito 5: abre uma caixa (reabre fechada ou cria nova).

Reutiliza uma caixa fechada se existir; se nao houver e ainda nao se atingiu N_CAIXAS, cria uma nova. Precisa de um operador livre. Os clientes ja em fila nas outras caixas NAO sao movidos – a nova caixa comeca vazia e recebe naturalmente os clientes que acabam as compras (via EscolherMelhorCaixa) e ainda nao chegaram ao LIMITE_FILA_CAIXA das antigas.

Retorna
1 se conseguiu, 0 caso contrario.

Requisito 5: abre uma caixa.
