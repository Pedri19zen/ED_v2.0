Pesquisa case-insensitive + prefixo (fallback para varrimento linear).

Primeiro tenta match exacto (rapido, via hash); se nao encontrar, faz um varrimento linear case-insensitive com prefixo. Devolve o primeiro cliente activo que comece pelas mesmas letras (qualquer caixa).
