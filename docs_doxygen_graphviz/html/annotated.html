Estruturas de dados
Lista das estruturas de dados com uma breve descrição:
 CCaixaUma caixa de atendimento
 CClienteUm cliente do supermercado
 CFilaFila (FIFO) com referencias para o inicio e o fim
 CFuncionarioUm funcionario / operador de caixa
 CHashingTabela de dispersao das caixas (chave = nome)
 CHashingNomesTabela de dispersao de nomes
 CListaClientes"Lista" de clientes (array) com indice de pesquisa O(1)
 CListaFuncionarios"Lista" de funcionarios implementada como array (pesquisa linear)
 CListaProdutos"Lista" de produtos (array) com indice de pesquisa O(1)
 CNoFilaNo da lista ligada da fila
 CNoHashNo da tabela de dispersao das caixas
 CNoHashNomeNo da tabela de dispersao nome -> indice
 CNoNomeNo simples para guardar os nomes dos clientes ja atendidos
 CProdutoUm produto do catalogo
 CRelogioRelogio logico que avanca em passos discretos
 CSupermercadoEstrutura central do projecto
